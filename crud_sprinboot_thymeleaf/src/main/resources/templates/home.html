Crud Básico

Este é um exemplo de um crud básico em Java utilizando a framework Spring Boot, banco de dados utiliza o docker-composer com a imagem do Mysql.

Os usuários possuem perfil de adm ou default, os usuários adm tem acesso a lista dos outros usuários, e pode filtra-los por status ou roles. O usuário padrão terá acesso somente ao seu perfil e a página home.

Os testes unitários foram feitos com Junit .
